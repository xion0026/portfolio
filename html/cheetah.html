Cheetah Networks Website Redesign

Redesigned a B2B telecom website to improve clarity, highlight key offerings, and create a more engaging user journey.

Design Engineer (UX/UI + Frontend Strategy) • 2024 • Figma, CSS Architecture, Responsive Systems

Background

Cheetah Networks needed to close the gap between their complex telecom infrastructure services and their diverse B2B clientele. I led the redesign to transform dense technical language into a high-converting visual narrative, ensuring the platform is not merely aesthetically modern but also technically optimized for worldwide accessibility.

Main Challenge

Users struggled to quickly understand the company’s core services and why they should trust the brand.

Problem

  • Cognitive Overload: The dense layout led to high drop-off rates as users struggled to navigate complex service layers.
  • Key services were not highlighted clearly enough.
  • Technical Debt & Low Engagement: Beyond aesthetics, the legacy structure lacked responsive agility and modern SEO standards.
  • The overall experience felt dated and low-engagement.

Design-to-Code

  • Scalable Component Library: Designed a modular UI kit in Figma to ensure a 1:1 hand-off, minimizing development friction.
  • Performance-First Design: Optimized all visual assets and layout structures to ensure fast TTI (Time to Interactive) across B2B network environments.
  • Responsive Grid Logic: Implemented a fluid grid system that maintains readability from 27-inch monitors to mobile devices.

My Approach

Simplified Architecture

Reorganized content so users could understand the business faster.

Clear Hierarchy

Used stronger spacing, headings, and grouping to improve readability.

Modular Layout

Introduced flexible card-based sections to support future content growth.

Process

Research

Competitor analysis, content audit, and review of the existing experience.

Ideation

Wireframes, section experiments, and layout exploration.

Design

High-fidelity UI, responsive layouts, and a cleaner visual system.

Final Design

A clear modular layout improves scanability and makes the value proposition easier to understand.

Key services are highlighted more clearly to support faster decision-making.

Before vs After

Before: cluttered layout with weaker content structure.

After: cleaner structure, stronger hierarchy, and a more modern presentation.

Results

  • Improved content clarity and messaging focus.
  • Better navigation flow for potential clients.
  • Stronger visual hierarchy across the site.
